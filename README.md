# Flask
Flask Framework으로 구현한 개인 웹 사이트

## 프로젝트 소개
* 이 프로젝트는 Micro Framework인 Flask를 활용한 개인 블로그를 구현한 프로젝트입니다.
* 회원가입 / 로그인 기능과 게시글 관리 기능이 구현되어 있습니다.

## 사용법
1. Flask 라이브러리를 로컬 혹은 가상환경에 설치합니다.
```python
pip install Flask
```
2. server.py를 실행시킵니다.
```python
python server.py
```

3. ***locahlost:8080/flask/main***로 접속합니다.
